What does Gameday Parking do?

Gameday Parking is a robust web-app meant to connect people looking for parking with people looking to lease.

Specifically, users will be able to navigate many parking availabilities and request/reserve parking spots through the app. Admins will be able to post available parking spots with details (amenities, size of space, grill availability, etc)

What we have done so far?

In the first iteration, we created necessary database models and create a basic web interface to populate thsoe modes.

What we will do in the near future?

In the second iteration, we will add additional features to the web interface and refine it, and reate web interfaces to edit database models.

In the thrid iteration, we will complete full test suite, and add reach goals that have no dependencies on them (e-commerce, mobile app, etc).
